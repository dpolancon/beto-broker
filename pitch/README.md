# Pitch: A Living System for Financial Market Self-Discovery

> Perplexity Computer Contest Submission — April 2026  
> **Reproducible Prompt v1.0 · No PII · Version-controlled**

---

## One-Sentence Thesis

A reproducible, AI-human interactive simulation that turns explicit research priors into a live, self-updating knowledge graph and score-weighted index — built entirely on Perplexity Computer, Obsidian, and a local GitHub repo, with no black boxes and no cloud dependency.

---

## Why This Matters

Most AI-assisted trading tools are either black boxes or require institutional infrastructure. This system proves that a single retail user with a thesis, a laptop, and a $100 normalized unit can build a fully auditable, self-discovering financial research environment — with Perplexity Computer as the engine that reads, reasons, and suggests at every step.
