# vectorbt

## What it teaches
Vectorized backtesting — runs thousands of parameter combinations simultaneously
using NumPy/pandas under the hood. Teaches signal robustness across parameter space
and identifies which scoring thresholds historically produced alpha.
Limitation: simplified fill model, no slippage or borrow cost simulation.
