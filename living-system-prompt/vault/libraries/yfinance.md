# yfinance

## What it teaches
Access to OHLCV data at 1m resolution, option chains, and basic fundamentals.
Primary lens for scoring inputs (EV/Sales, P/FCF proxies) and PII returns.
Limitation: delayed data, no streaming tick feed, no Level 2.
Switch to openbb when multi-source or alternative data is needed.
